[Note] Interlaced Sparse Self-Attention for Semantic Segmentation (IJCV 2021)

发表于 2023-07-25 分类于 学习 本文字数： 1.2k 阅读时长 ≈ 4 分钟

作者：Lang Huang, Yuhui Yuan, Jianyuan Guo, Chao Zhang, Xilin Chen, Jingdong Wang

ISANet

本文提出了交错稀疏注意力机制（Interlaced Sparse Self-Attention），目的在于提高自注意力机制的计算效率和内存占用。交错稀疏注意力机制将稠密的注意力矩阵（Affinity）分解为两个注意力矩阵，分别用于计算长程的注意力和短程的注意力。这一方法极大地降低了自注意力机制的计算/内存成本，对于各种视觉任务有较大的应用价值。

  • 长程注意力：估计具有长空间间隔的子集内的注意力
  • 短程注意力：估计具有短空间间隔的子集内的注意力

[Note] RefineNet: Multi-Path Refinement Networks for High-Resolution Semantic Segmentation (CVPR 2017 & TPAMI)

发表于 2023-07-24 分类于 学习 本文字数： 597 阅读时长 ≈ 2 分钟

作者：Guosheng Lin, Anton Milan, Chunhua Shen, Ian Reid

RefineNet

本文介绍了一个多路径细化（multi-path refinement）的网络RefineNet，使用长程的残差连接来实现语义分割中的高分辨率预测。该网络利用了在下采样过程中可用的所有信息，使用浅层卷积的细粒度特征来细化更深层次高级语义特征。另外，文章中引入了链式残差池化（chained residual pooling），以有效地捕获丰富的背景上下文。

常用的深度CNN网络（VGG，ResNet），由于池化和卷积的存在，特征图会越来越小，导致损失一些细粒度的信息。浅层的特征图有丰富的细粒度信息，深层的特征图则有更抽象、粗粒度的信息。对于分类问题而言，只需要深层的语义信息就能表现较好，但是对于稠密预测问题（语义分割、深度估计），除了需要强语义信息之外，还需要高空间分辨率。

常用的解决方法：

  • 反卷积：不能恢复低级的空间信息。
  • 空洞卷积：①计算量大（维持高分辨率特征图）；②是对特征图的一种粗粒度采样，会丢失重要信息。
  • 引入中间层特征：文中说丢失了强空间信息。在当时还没出现FPN、UPerNet等工作。

[Note] Adaptive Pyramid Context Network for Semantic Segmentation (CVPR 2019)

发表于 2023-07-22 更新于 2023-07-21 分类于 学习 本文字数： 1.1k 阅读时长 ≈ 4 分钟

作者：Junjun He, Zhongying Deng, Lei Zhou, Yali Wang, Yu Qiao

APCNet

本文提出了基于自适应金字塔上下文的语义分割网络（APCNet），用于解决语义分割任务中上下文信息构建的问题（如何找到最好的上下文信息？）。

文中提到理想的上下文信息应该有的特征：

  • 多尺度（Multi-scale）：由于对象通常具有不同的大小和位置，因此有必要构建多尺度表示来捕获不同尺度的图像内容。没有多尺度上下文的方法只能捕获单一尺度的对象，并且会在其他尺度上丢失细节。
  • 自适应（Adaptive）：输入图像中的所有区域并不都等同地决定给定像素的语义标签。包含相关对象的区域可以提供有用的信息，而其他区域可能贡献很少。自适应地识别这些重要区域以构建最优的上下文向量是很重要的。
  • 全局信息引导的局部关联性权重（Global-guided Local Affinity，GLA）：为了构建有效的上下文向量，需要聚合来自相关像素或区域的特征。在实践中，可以通过加权和的方式实现。合理的权重需要同时考虑局部信息和全局信息。
    • Affinity：用来描述不同区域对预测局部像素的语义标签的贡献程度。

[Note] DeepLab: Semantic Image Segmentation with Deep Convolutional Nets, Atrous Convolution, and Fully Connected CRFs (TPAMI)

发表于 2023-07-04 更新于 2023-07-10 分类于 学习 本文字数： 2.5k 阅读时长 ≈ 9 分钟

作者：Liang-Chieh Chen, George Papandreou, Iasonas Kokkinos, Kevin Murphy, Alan L. Yuille

DeepLabv2

DeepLabv2通过空洞卷积和上采样滤波器进行密集特征提取，将在图像分类上训练的网络重新用于语义分割任务。文中进一步提出ASPP以在多个尺度上编码对象以及图像上下文。为了产生语义准确的预测和精细的物体边界分割图，文中还结合了深度卷积神经网络和全连接条件随机场的思想。

深度卷积神经网络应用于语义分割的三个挑战：

  • 特征分辨率降低：DCNN设计之初用于图像分类问题，使用多层的Max-Pool和全连接层，在处理图像时，特征空间分辨率降低。

  • 多尺度目标：图像中的分割对象以多种尺度存在。

  • DCNN的不变性导致的定位误差：深度卷积神经网络的不变性可能会导致定位精度降低。

为了解决这些问题，DeepLabv2使用了下面的方法：

  • 去除Max-Pool层并使用空洞卷积（atrous convolution）
  • 空洞空间金字塔池化（atrous spatial pyramid pooling，ASPP）
  • 深度卷积网络与概率图模型结合，使用全连接条件随机场（fully connected CRFs）进行后处理

[Note] Coordinate Attention for Efficient Mobile Network Design (CVPR 2021)

发表于 2023-05-12 更新于 2023-05-14 分类于 学习 本文字数： 1.7k 阅读时长 ≈ 6 分钟

作者：Qibin Hou, Daquan Zhou, Jiashi Feng

Coordinate Attention

本文主要介绍了一种针对轻量级网络的坐标注意力机制（Coordinate Attention）。与通道注意力不同，坐标注意力从\(H\)和\(W\)两个方向聚合特征，这样在一个方向上获得了长程依赖关系（long-range dependencies），在另一个方向上保留了精确的位置关系。坐标注意力机制可以简单灵活地插入到经典的轻量级网络中，例如MobileNetV2、MobileNeXT和EfficientNet中，几乎不需要计算开销。通过在ImageNet分类、目标检测和语义分割等任务上进行实验，证明了该机制相比其他注意力机制具有更好的性能表现。

背景

注意力机制用于告诉模型“在哪里（What）”和“关注什么（Where）”。考虑到轻量级网络的计算限制，SE（Squeeze-and-Excitation）注意力机制仍然是最流行的方法，它利用2D全局池化计算通道注意力。SE通过squeeze，excitation和scale操作得到\(c\)个权重系数，用来自适应地标定\(c\)个通道。在计算成本相对较低的情况下，SE注意力提供显着的性能提升。

然而，SE注意力只考虑了通道之间的信息，而忽略了位置信息的重要性。之后的工作，如BAM和CBAM，通过减少输入张量的通道维度，然后使用卷积计算空间注意力来利用位置信息。但是，卷积只能捕获局部关系，不能建模对于视觉任务至关重要的长程依赖关系。

本文在之前的工作基础上，提出了一种新颖且高效的注意力机制，通过将位置信息嵌入通道注意力中，使得网络可以关注大范围的区域，同时避免产生显著的计算开销。为了减轻2D全局池化引起的位置信息丢失，本文将通道注意力分解为两个并行的1D特征编码过程，以有效地将空间坐标信息整合到生成的注意力映射中。具体来说，我们的方法利用两个1D全局池化操作，分别沿着垂直和水平方向聚合输入特征，生成两个分别包含方向特定信息的特征映射。然后，将这两个特征映射分别编码成两个注意力映射，每个注意力映射都捕获输入特征映射沿一个空间方向的长程依赖关系，从而能够保留位置信息。然后，通过乘法将两个注意力映射应用于输入特征映射，以强调感兴趣的特征。
