[Note] TOOD: Task-aligned One-stage Object Detection (ICCV 2021)

发表于 2023-04-21 更新于 2023-04-22 分类于 学习 本文字数： 2.8k 阅读时长 ≈ 10 分钟

作者：Chengjian Feng, Yujie Zhong, Yu Gao, Matthew R. Scott, Weilin Huang

TOOD

背景

单阶段的目标检测算法通常通过优化两个子任务——分类（object classification）和定位（localization）来实现。分类任务的目标是学习物体具有区分力的特征，定位任务的目标是对物体的边界进行准确定位。这两个任务使用两个并行的分支进行，因而可能会导致两个任务之间的预测存在一定程度的空间不对齐（spatial misalignment）。

上图中第一行是ATSS1预测的Dining table，第二行是TOOD预测的Dining table。第二列是预测结果的分类（Score）置信度，第三列是定位（IoU）的置信度。从ATSS的预测结果看，Score最高的位置，并不是定位最准确的位置。这样的情况，可能会导致在NMS的时候，预测更准的边界框会被过滤掉。

现有的单阶段方法通过目标的中心点来试图实现两个任务的统一，这些方法假设在物体中心点处的锚框（或锚点）能给出更准确的预测结果。例如FCOS2和ATSS3都使用中心性（centerness）分支来提高在物体中心附近的锚框（或锚点）的分类得分，并且给对应的锚框（或锚点）的定位损失更多的权重。但是，这些方法存在两个问题：

  1. 分类和定位是独立的：这种双分支设计可能会导致两个任务之间缺乏交互，从而导致预测不一致。
  2. 与任务无关（Task-agnostic）的样本分配：大多数的无需锚框的方法使用基于几何距离的样本分配方法，基于锚框的方法使用基于IoU的样本分配方法。然而，用于分类和定位的最佳锚框（或锚点）通常不一致，并且可能会根据对象的形状和特征而有很大差异。常用的与任务无关的样本分配方法难以对两个任务进行准确而一致的预测。

本文提出任务对齐的单阶段目标检测（TOOD）方法，通过学习的方式自动对齐两个任务。

  • Task-Aligned Head（T-Head）

T-Head在学习用于任务交互的特征（task-interactive features）和任务特定的特征（task-specific features）之间提供了更好的平衡，并且通过任务对齐的预测器（Task-Aligned Predictor）在模型学习中达到更高的灵活性。

  • Task-Aligned Learning（TAL）

为了进一步解决不对齐问题，TAL通过设计的样本分配方案和任务对齐损失，在训练期间明确地拉近（甚至统一）两个任务的最优锚点。样本分配通过计算每个锚点的任务对齐程度来收集训练样本（即正样本或负样本），而任务对齐损失在训练期间逐渐统一最佳锚点，以预测分类和定位。因此，在推理时，可以保留具有最高分类分数和最精确定位的边界框。

[Note] DETRs Beat YOLOs on Real-time Object Detection

发表于 2023-04-20 更新于 2023-04-21 分类于 学习 本文字数： 2.3k 阅读时长 ≈ 8 分钟

作者：Wenyu Lv, Shangliang Xu, Yian Zhao, Guanzhong Wang, Jinman Wei, Cheng Cui, Yuning Du, Qingqing Dang, Yi Liu

RT-DETR

背景

近来，基于Transformer的端到端目标检测方法（DETR）取得了非常好的效果，但它同时也存在计算量大的问题，因此很少在实时场景下应用，这也进一步阻止了我们充分利用它无需后处理（NMS）的优势。

本文介绍了一种新的实时目标检测检测器RT-DETR，在推理速度和准确性方法达到最优，并且不需要后处理（NMS）。文中分析了NMS对于目标检测算法推理速度的影响，并建立了一个端到端的速度测试基准，指出Anchor-Free的方法更具优势。为了避免NMS引起的推理延迟，本文使用了DETR进行端到端目标检测，通过提出高效的混合编码器来处理多尺度特征和IoU感知的query选择，来实现高性能的实时目标检测。

RT-DETR-L在COCO val2017上达到53.0％AP，在NVIDIA Tesla T4 GPU上达到114FPS；而RT-DETR-X则达到54.8％AP和74FPS，在速度和准确性方面均优于同等规模的所有YOLO检测器。

基于NeRF的三维场景生成

发表于 2023-04-18 更新于 2023-04-20 分类于 学习 本文字数： 3.3k 阅读时长 ≈ 12 分钟

NeRF = Neural Radiance Fields

背景

经典三维场景表征与神经表征

经典的三维场景表征方法有体素表示、点云表示和网格表示，这三种表示是直接的、显而易见的，因此归为显式的场景表示类别。这里介绍的NeRF（Neural Radiance Fields）其实也是一种三维场景表征，但是是一种隐式的场景表示（implicit scene representation），因为它不能像点云、网格、体素一样直接看见一个三维模型，需要将神经表征转换到显示的表征或渲染成可见的图像才可以被看到。

显示场景表征的特点是，它表示的信号是离散的。相反，隐式神经表征将信号参数化为连续函数，通常来说，普通的函数不可能做到这样一件事，因此用神经网络来逼近这样一个复杂的“自然函数”。

  • 显示场景表征——离散

  • 隐式场景表征——连续

经典的SFM重建方法

  1. 基于输入图像对相应特征进行提取，并根据特征相似性进行匹配；使用一些几何先验性知识，对匹配结果进行确认。这个过程是特征匹配的过程。经过以上步骤之后可以获得相机的位姿。
  2. 使用相机位姿投影并经过图像配准、三角化测量等步骤反复迭代，可以获得特征点的三维空间坐标。
  3. 再经过bundle adjustment（BA）和噪声过滤的操作，便可以获得三维重建的最终结果。

NeRF方法的思想

使用Nerf进行重建，也是需要相机姿态的，这一步和colmap没有区别。

NeRF方法的主要想法通过输入稀疏的图像集，来优化底层的连续神经网络，该神经网络将三维场景隐式存储其中，我们只需要通过输入一个相机位姿，就可以获得场景图片，实现了合成复杂场景的全新视图。

输入图像的位姿是已知的，这一位姿既可以通过外部测量的方式获取，也可以、也是最常用的方法是使用SFM（运动恢复结构）确定每张图像对应相机的外方位元素。

[Note] Segment Anything

发表于 2023-04-14 更新于 2023-04-16 分类于 学习 本文字数： 1.9k 阅读时长 ≈ 7 分钟

作者：Alexander Kirillov, Eric Mintun, Nikhila Ravi, Hanzi Mao, Chloe Rolland, Laura Gustafson, Tete Xiao, Spencer Whitehead, Alex Berg, Wan-Yen Lo, Piotr Dollar, Ross Girshick

SAM

SAM（Segment Anything Model）是Meta AI在今年4月5日发布的图像分割工作，具有非常炸裂的效果。SAM是用于图像分割的基础模型（Foundation Model），通过Prompt的方式，SAM模型具备非常强大的zero-shot和few-shot能力，能够直接迁移到没见过的图像分布和视觉任务中。项目地址：https://segment-anything.com，通过Demo可以感受到SAM的强大效果。

背景

在大规模Web数据集上预训练的大语言模型（LLM）通过提示工程（Prompt Engineering）的方式，在超出训练中所见任务和数据分布的领域具有极强的泛化能力（零样本或少样本）。经验表明，这种表现会随着模型规模、数据集大小以及总的训练计算量的增加而不断提高。

基础模型在计算机视觉领域也得到了一定的探索，虽然规模相对较小。CLIP和ALIGN使用对比学习框架训练文本和图像编码器来配对这两种模态数据，这些编码器进而可以与其他下游任务结合，如图像生成（DALL·E）等。虽然视觉与语言编码器取得了很多进展，但是计算机视觉包括许多超出这个范围的问题，并且对于这些问题，缺乏丰富的训练数据。

SAM构建了用于图像分割的视觉基础模型，通过大量的数据和Prompt的学习方式，具有强大的泛化能力。SAM的成功取决于三个组成部分：任务（Task）、模型（Model）和数据（Data）。

[Note] Run, Don’t Walk: Chasing Higher FLOPS for Faster Neural Networks (CVPR 2023)

发表于 2023-04-14 分类于 学习 本文字数： 1.4k 阅读时长 ≈ 5 分钟

作者：Jierun Chen, Shiu-hong Kao, Hao He, Weipeng Zhuo, Song Wen, Chul-Ho Lee, S.-H. Gary Chan

FasterNet

问题

研究者通常使用计算复杂度（通常表示为FLOPs）神经网络的推理速度，例如MobileNets、ShuffleNets和GhostNets等都利用了DWConv或GConv来降低计算复杂度。然而，在减少FLOPs的同时，这些算子带来了增加内存访问的副作用。

文中通过实验检验了延迟（Latency）和计算复杂度（FLOPs）之间的关系，说明了FLOPs更小的模型不一定更快。仅仅减少FLOPs并不一定会导致延迟的以同样程度降低，这主要是由于低效的每秒浮点运算次数（FLOPS）造成的。 \[ Latency=\frac{FLOPs}{FLOPS} \]

本文为了解决DWConv算子频繁内存访问的问题，提出了部分卷积（partial convolution，PConv）算子，达到同时减少计算冗余度和内存访问的目的。并介绍了基于PConv的FasterNet模型，它在各种设备上都能够快速运行。

[Note] A Simple Framework for Contrastive Learning of Visual Representations (ICML 2020)

发表于 2023-04-14 分类于 学习 本文字数： 1.3k 阅读时长 ≈ 5 分钟

作者：Ting Chen, Simon Kornblith, Mohammad Norouzi, Geoffrey Hinton

SimCLR

本文提出一个用于对比学习视觉表征的框架SimCLR。该框架不仅优于以前的工作，而且更简单，既不需要专门的体系结构，也不需要存储库（memory bank）。作者系统地研究了该框架的主要组成部分，并发现数据增强、可学习非线性变换（MLP）和批量大小、训练次数等因素对于有效学习视觉表征至关重要。本文提供了一种自我监督学习算法，可以在没有标签数据的情况下从未标记的图像中学习有用的特征表示。

SimCLR框架

受最近对比学习算法的启发，SimCLR通过在潜在空间中使用对比损失来最大化同一数据示例的不同增强视图之间的一致性来学习视觉表征。

SimCLR框架主要有以下几个部分组成：

  1. 随机数据增强得到同一样本的两个不同视角\(\tilde{\boldsymbol{x}}_{i}\)和\(\tilde{\boldsymbol{x}}_{j}\)，作为正样本对。在SimCLR中，按顺序应用了三种数据增强方法。作者通过消融实验证明了这三种数据增强方法的组合能取得最好的效果。
    1. 随机裁剪后缩放回原尺寸（random cropping）
    2. 随机色彩失真（random color distortions）
    3. 随机高斯模糊（random Gaussian blur）
  2. 基础的编码器\(f(\cdot)\)抽取图像表征。在SimCLR中使用了ResNet，即\(\boldsymbol{h}_{i} = f(\tilde{\boldsymbol{x}}_{i}) = \text{ResNet}(\tilde{\boldsymbol{x}}_{i})\)，其中\(\boldsymbol{h}_{i} \in \mathbb{R}^{d}\)。\(\boldsymbol{h}_{i}\)为ResNet中经过全局池化后的特征，即\(d = 2048\)。
  3. 投影头（projection head）\(g(\cdot)\)对特征进行非线性变换。在SimCLR中使用了两层的MLP（FC->ReLU->FC），即\(\boldsymbol{z}_{i} = g(\boldsymbol{h}_{i}) = W^{(2)} \sigma \left(W^{(1)}\boldsymbol{h}_{i}\right)\)，其中\(\sigma\)为ReLU函数。
  4. 对比损失函数定义了对比学习任务。给定一个集合\(\{\tilde{\boldsymbol{x}_k}\}\)，其中包含一个正样本对\(\tilde{\boldsymbol{x}}_{i}\)和\(\tilde{\boldsymbol{x}}_{j}\)，对比学习任务旨在当给定\(\tilde{\boldsymbol{x}}_{i}\)时，从集合\(\{\tilde{\boldsymbol{x}_k}\}_{k \ne i}\)中识别出\(\tilde{\boldsymbol{x}}_{j}\)。

[Note] Human Pose Regression with Residual Log-likelihood Estimation (ICCV 2021)

发表于 2023-04-13 分类于 学习 本文字数： 2.6k 阅读时长 ≈ 10 分钟

作者：Jiefeng Li, Siyuan Bian, Ailing Zeng, Can Wang, Bo Pang, Wentao Liu, Cewu Lu

RLE（使用残差对数似然估计的人体姿态回归）

本文主要介绍了一种新的人体姿态估计方法，该方法使用Residual Log-likelihood Estimation（RLE）来捕捉输出分布的内在特征，以提高回归性能。文章指出标准的\(\mathcal{l}_1\)和\(\mathcal{l}_2\)损失函数在人体姿态估计任务中存在一些问题，如无法捕捉复杂的输出分布、对异常值敏感、难以处理复杂分布等。为了解决这些问题，文章提出了一种基于RLE的新型回归范式，并将其与Normalizing Flows相结合，以更好地捕获复杂的输出分布。文中通过实验验证了该方法在各种人体姿态估计任务中的有效性和灵活性。

人体姿态估计问题

姿态估计任务中，一般存在两种方法：

  • 热力图（heatmap-based）
  • 坐标回归（regression-based）

基于热力图的方法通过监督模型学习到高斯概率分布图，网络输出的\(K\)张热力图对应了\(K\)个关键点，通过argmax来取得最大值坐标作为关键点的预测。这种方法不可避免地需要输出多个相对高分辨率的输出，自然导致了很大的计算量和内存开销（尤其是从二维向三维扩展时）。但是由于基于热力图的方法具有相当好的表现，仍然是目前主流的方法。

基于回归的方法直接预测坐标值，通过计算\(\mathcal{L}_1\)或\(\mathcal{L}_2\)损失函数进行优化，不需要高分辨率输出，也可以轻松向高维扩展。然而回归方法相比热力图方法表现较差，在图像出现遮挡、模糊和截断等情况下，不具备热力图方法的鲁棒性。另外，回归方法也容易受到标注中的噪声影响。

尽管基于回归的方法仍存在预测精度上的不足（这也是本文提出RLE要解决的问题），回归方法可以总结出以下三点优势：

  • 摆脱了高分辨率热力图，计算成本和内存开销的复杂度低。
  • 输出是连续的，不受量化误差影响。
  • 可以以最低的成本扩展到各种应用场景（例如，单阶段方法、基于视频的方法、3D 场景）。

回归方法在实际项目应用中比热力图方法更具有可行性。
