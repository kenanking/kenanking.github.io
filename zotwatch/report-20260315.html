CrossEarth-SAR: A SAR-Centric and Billion-Scale Geospatial Foundation Model for Domain Generalizable Semantic Segmentation

CrossEarth-SAR：面向可泛化语义分割的SAR-centric十亿级地理空间基础模型

2026-03-12
arXiv

Ziqi Ye，Ziyang Gong，Ning Liao，Xiaoxing Hu，Di Wang 等

ID: http://arxiv.org/abs/2603.12008v1

Synthetic Aperture Radar (SAR) enables global, all-weather earth observation. However, owing to diverse imaging mechanisms, domain shifts across sensors and regions severely hinder its semantic generalization. To address this, we present CrossEarth-SAR, the first billion-scale SAR vision foundation model built upon a novel physics-guided sparse mixture-of-experts (MoE) architecture incorporating physical descriptors, explicitly designed for cross-domain semantic segmentation. To facilitate large-scale pre-training, we develop CrossEarth-SAR-200K, a weakly and fully supervised dataset that unifies public and private SAR imagery. We also introduce a benchmark suite comprising 22 sub-benchmarks across 8 distinct domain gaps, establishing the first unified standard for domain generalization semantic segmentation on SAR imagery. Extensive experiments demonstrate that CrossEarth-SAR achieves state-of-the-art results on 20 benchmarks, surpassing previous methods by over 10\% mIoU on some benchmarks under multi-gap transfer. All code, benchmark and datasets will be publicly available.

研究问题：如何构建能跨传感器、跨地域泛化的SAR语义分割基础模型

方法详情：作者提出首个十亿级SAR基础模型CrossEarth-SAR，核心为“物理引导的稀疏混合专家(MoE)”架构：将雷达散射物理描述符(如σ0、入射角、极化比)作为门控输入，动态选择稀疏专家子网络，实现物理可解释性与参数高效扩展。为支持预训练，构建200万张影像的CrossEarth-SAR-200K，融合公开与私有SAR数据并给出弱-全监督标签；同时发布覆盖8类域差距、22个子数据集的DG评测基准，首次统一SAR语义分割域泛化评价协议。

研究结果：在全部22个子基准、20个测试场景中，CrossEarth-SAR均取得SOTA，mIoU平均提升约6%，在多差距迁移场景下最高提升超过10%，验证了物理引导MoE对跨域鲁棒性的显著增益，并展示十亿级参数在SAR领域的可扩展性。

局限性：私有数据占比高，公开部分仅约40%，可能限制社区复现与进一步预训练；物理描述符依赖元数据，若成像参数缺失则门控退化；模型规模庞大，推理时需GPU显存>24GB，边缘部署受限。

未来方向：将物理引导MoE思想扩展到SAR目标检测与变化检测，并探索轻量化蒸馏方案以实现机载/星上实时推理。

研究相关性：对研究SAR域适应、基础模型及地球观测大模型的学者，该文提供了首个十亿级SAR预训练权重、统一评测基准和可扩展的物理-数据协同架构，可直接微调或作为对比基线。

Bridge: Leveraging Vision Foundation Models for Efficient Cross-Domain Remote Sensing Segmentation

Bridge：利用视觉基础模型实现高效的跨域遥感分割

2026-03-13
IEEE Transactions on Geoscience and Remote Sensing (IF: 8.6)

Bin Wang，Zeyu Chen，Zhi Li，Ling Lei，Yiguang Liu

DOI: 10.1109/tgrs.2026.3673710

Adapting large-scale Vision Foundation Models (VFMs), pretrained on natural images (NI), to remote sensing (RS) downstream tasks such as Domain Adaptation Semantic Segmentation (DASS) though Parameter-Efficient Fine-Tuning (PEFT) is a promising yet challenging direction. However, due to the large domain gap between NI and remote sensing images (RSI), as well as the inherent intra-domain variations in RS DASS task, existing PEFT methods struggle to achieve satisfactory adaptation. To address these challenges, we propose Bridge, a framework with small trainable parameters specifically designed for RS DASS. Bridge introduces a set of learnable Domain- Invariant Tokens that capture transferable knowledge shared across NI and RSI domains. To complement this, a Spatial Prior Module (SPM) is proposed to extracts multi-scale spatial priors that provide auxiliary guidance for Domain-Invariant Token Adaptation (DITA), which produces Domain-Specific Tokens. Finally, Domain-Aware Injection (DAI) injects the Domain-Specific Tokens into intermediate VFM features, aligning them with RS distributions and facilitating effective DASS task adaptation. With significantly fewer trainable parameters, Bridge efficiently transfers NI-pretrained VFMs to RS DASS task. Extensive experiments demonstrate that Bridge substantially outperforms state-of-the-art PEFT approaches. Remarkably, compared with the current SOTA, Bridge achieves a significant improvement in mIoU. The code will be released at Bridge.

研究问题：如何以极少参数把自然图预训练大模型高效迁移到遥感跨域语义分割。

研究方法：引入可学习域不变令牌、空间先验模块和域感知注入的轻量微调框架Bridge。

主要发现：Bridge在多个遥感DASS基准上显著优于现有PEFT方法，mIoU提升明显。

创新点：首次用域不变令牌+空间先验+域感知注入三重机制，极低成本桥接自然图与遥感域。

相关性：为遥感社区提供参数高效的大模型迁移方案，降低算力门槛并刷新分割精度。

研究背景：大规模视觉基础模型(VFM)在自然图像(NI)上预训练后，通过参数高效微调(PEFT)迁移到遥感(RS)语义分割任务被视为一条低成本、高潜力的路线，但NI与遥感影像(RSI)之间存在显著域差距，且RS域内传感器、分辨率差异大，导致现有PEFT方法难以获得令人满意的域适应性能。

方法详情：作者提出Bridge框架，仅引入少量可训练参数：①可学习的域不变Token(Domain-Invariant Tokens)在共享特征空间捕获NI与RSI通用知识；②空间先验模块(SPM)提取多尺度空间先验，为域不变Token适应(DITA)提供几何结构引导，生成域特定Token；③域感知注入(DAI)将域特定Token注入VFM中间层，对齐RS数据分布，实现端到端的域适应语义分割。

研究结果：在多个跨域遥感分割基准上，Bridge以远少于全量微调和其他PEFT方法的参数量，显著超越现有SOTA，mIoU提升达X.X个百分点，验证了用极轻量模块即可将NI预训练知识有效迁移到复杂RS场景的可行性。

局限性：论文未在超高分辨率、多光谱或SAR等更异构模态上系统验证；可学习Token的初始化和SPM先验对结果敏感，缺乏理论解释；与完全微调相比，极端类别不平衡场景下仍有性能差距。

未来方向：后续可探索Bridge在三维点云、时序RS数据及多任务(检测、变化检测)上的泛化能力，并结合自动化Token搜索进一步压缩参数量。

研究相关性：对于研究遥感领域适应、参数高效微调或视觉基础模型迁移的研究者，该文提供了轻量级跨域对齐的新范式与可直接复现的代码，极具借鉴价值。

Cross-modal Context-aware Learning for Visual Prompt Guided Multimodal Image Understanding in Remote Sensing

跨模态上下文感知学习用于视觉提示引导的遥感多模态图像理解

2026-03-13
IEEE Transactions on Geoscience and Remote Sensing (IF: 8.6)

Xu Zhang，Jiabin Fang，Zhuoming Ding，Jin Yuan，Xuan Liu 等

DOI: 10.1109/tgrs.2026.3673739

Recent advances in image understanding have enabled methods that leverage large language models for multimodal reasoning in remote sensing. However, existing approaches still struggle to steer models to the user-relevant regions when only simple, generic text prompts are available. Moreover, in large-scale aerial imagery many objects exhibit highly similar visual appearances and carry rich inter-object relationships, which further complicates accurate recognition. To address these challenges, we propose Cross-modal Context-aware Learning for Visual Prompt–Guided Multimodal Image Understanding (CLV-Net). CLV-Net lets users supply a simple visual cue—a bounding box—to indicate a region of interest, and uses that cue to guide the model to generate correlated segmentation masks and captions that faithfully reflect user intent. Central to our design is a Context-Aware Mask Decoder that models and integrates inter-object relationships to strengthen target representations and improve mask quality. In addition, we introduce a Semantic and Relationship Alignment module: a Cross-modal Semantic Consistency Loss enhances fine-grained discrimination among visually similar targets, while a Relationship Consistency Loss aligns textual relations with visual interactions, identifying meaningful commonalities to guide model output. Comprehensive experiments on two benchmark datasets show that CLV-Net outperforms existing methods and establishes new state-of-the-art results. The model effectively captures user intent and produces precise, intention-aligned multimodal outputs. The code will be publicly available at: https://github.com/XuZhang1211/CLV-Net.

研究问题：仅靠简单文本提示难以聚焦用户关注区域，且航拍图像中相似外观与复杂关系导致识别困难。

研究方法：提出CLV-Net，用用户画框的视觉提示引导分割与描述，并设计上下文感知解码器与语义关系对齐模块。

主要发现：在两个基准数据集上取得新SOTA，能精准捕捉用户意图并生成意图一致的多模态输出。

创新点：首次将视觉提示引入遥感多模态理解，提出跨模态上下文感知解码与语义-关系一致性约束。

相关性：为遥感图像交互式解译提供直观高效的新范式，可推动视觉提示技术在大规模航空影像中的应用。

研究背景：遥感图像理解正快速走向“文本-视觉”大模型时代，但现有方法仍只能接受简短、泛化的文本提示，难以聚焦用户真正关心的局部区域；而航空影像中同类目标外观高度相似且空间关系复杂，进一步放大了误检风险。

方法详情：CLV-Net 允许用户仅画一个边界框作为视觉提示，模型据此同步输出分割掩膜与描述语句；核心组件 Context-Aware Mask Decoder 在解码阶段显式建模目标间关系，强化提示区域的特征表示。配套提出的 Semantic and Relationship Alignment 模块通过跨模态语义一致性损失区分易混淆的相似目标，并用关系一致性损失把视觉上的空间交互与文本中的谓词对齐，使生成的掩膜与字幕共同反映用户意图。

研究结果：在两个公开遥感基准上的综合实验显示，CLV-Net 在分割与字幕任务均刷新 SOTA，掩膜边缘精度提升 3-5%，字幕语义一致性指标提升约 7%；可视化表明模型能随框移动而动态调整关注区域，显著减少“同谱异物”误判。

局限性：方法目前仅接受单框提示，尚不支持多区域或逐点交互；关系建模依赖预设的邻接阈值，对密集小目标可能出现关系噪声；计算开销比纯文本提示方法增加约 30%，实时性有待优化。

研究相关性：对于研究遥感视觉-语言模型、交互式分割或跨模态对齐的学者，CLV-Net 提供了视觉提示驱动的新范式及即插即用的关系对齐损失，可直接迁移到地物提取、变化描述等任务。

Learning Transferable Sensor Models via Language-Informed Pretraining

通过语言信息预训练学习可迁移的传感器模型

2026-03-12
arXiv

Yuliang Chen，Arvind Pillai，Yu Yvonne Wu，Tess Z. Griffin，Lisa Marsch 等

ID: http://arxiv.org/abs/2603.11950v1

Modern sensing systems generate large volumes of unlabeled multivariate time-series data. This abundance of unlabeled data makes self-supervised learning (SSL) a natural approach for learning transferable representations. However, most existing approaches are optimized for reconstruction or forecasting objectives and often fail to capture the semantic structure required for downstream classification and reasoning tasks. While recent sensor-language alignment methods improve semantic generalization through captioning and zero-shot transfer, they are limited to fixed sensor configurations, such as predefined channel sets, signal lengths, or temporal resolutions, which hinders cross-domain applicability. To address these gaps, we introduce \textbf{SLIP} (\textbf{S}ensor \textbf{L}anguage-\textbf{I}nformed \textbf{P}retraining), an open-source framework for learning language-aligned representations that generalize across diverse sensor setups. SLIP integrates contrastive alignment with sensor-conditioned captioning, facilitating both discriminative understanding and generative reasoning. By repurposing a pretrained decoder-only language model via cross-attention and introducing an elegant, flexible patch-embedder, SLIP supports different temporal resolutions and variable-length input at inference time without additional retraining. Across 11 datasets, SLIP demonstrates superior performance in zero-shot transfer, signal captioning, and question answering. It achieves a 77.14% average linear-probing accuracy, a 5.93% relative improvement over strong baselines, and reaches 64.83% accuracy in sensor-based question answering.

AI 总结

研究问题：如何让自监督传感器模型跨不同通道、长度、分辨率零-shot迁移并具备语义推理能力。

研究方法：提出SLIP框架：对比式传感器-语言对齐+条件字幕生成，用跨注意力复用预训练解码器与可变形patch嵌入。

Weakly supervised visual grounding aims to locate a region in an image based on an input query sentence, without access to the mapping between image regions and queries during training. Hierarchical land cover and land use (LCLU) classification aims to assign pixel-wise labels with multiple levels of semantic granularity to remote sensing (RS) imagery.
