A Guided Fusion Network based on Cross-Scale Semantic Alignment for multi-spectral object detection

基于跨尺度语义对齐的引导融合网络用于多光谱目标检测

2026-05-23
Pattern Recognition (IF: 7.6)

Hongtao Wu，Jing Rong，Qiuzhan Zhou，Di Yang，Weiqi Wang

DOI: 10.1016/j.patcog.2026.114024

Multi-spectral object detection gains significant attention due to its potential in all-weather applications, particularly those involving RGB and Infrared (IR) images. However, existing methods fail to account for the semantic differences between the RGB and IR modalities, resulting in insufficient modal feature fusion and limiting the performance of RGB-IR object detection. To address these issues, we propose a novel learning network, the Semantic Alignment Guided Fusion Network (SAGF-Net), that fuses RGB and IR images in a guided manner to extract more complementary feature representations. Our network consists of three key modules: Cross-Scale Semantic Alignment (CSSA), Cross-Modal Information Guidance (CMIG), and Deformable Cross-Attention (DCA). The CSSA module alleviates semantic differences across different receptive fields through cross-scale dynamic attention fusion. Subsequently, the CMIG module extracts modality-specific advantageous features and guides the other modality to enhance the expression of key information, obtaining refined and enhanced feature representations. To reduce computational complexity, we introduce the DCA module, which selects only partial keys and values for each query within a local region of the feature map. Extensive comparative experiments and ablation experiments on multiple public datasets achieve state-of-the-art performance, demonstrating the effectiveness of the proposed method.

PolSAR vehicle recognition via scattering mechanism-driven hybrid attention

基于散射机制驱动的混合注意力PolSAR车辆识别

2026-05-23
Pattern Recognition (IF: 7.6)

Jie Deng，Wei Wang，Huiqiang Zhang，Deliang Xiang，Jun Zhang

DOI: 10.1016/j.patcog.2026.114061

Polarimetric synthetic aperture radar (PolSAR), as an advanced remote sensing technology, can provide rich polarization information and is of great significance for target recognition in complex environments. However, deep learning methods based on neural networks often fall into the dilemma of convergence or over-fitting due to insufficient samples and weak physical perception. To address this limitation, we propose a PolSAR vehicle target recognition method via scattering mechanism-driven hybrid attention. Firstly, we introduce a physics-inspired neural module, termed the Polarization Knowledge Extraction Module (PKEM), which retrieves physical scattering information from the Discrete Null-Pol Synthesis Pattern (DNSP). We then propose a Polarization Guided Feature Enhancement (PGFE) module, which dynamically re-weights deep features according to the physical prior information, thereby enhancing the model’s physical perception capabilities. Finally, we constructed a PolSAR vehicle target recognition dataset that can significantly facilitate the advancement of PolSAR ATR. Extensive experimental results on our self-collected measured dataset and Gotcha dataset confirm the superior effectiveness of the proposed physics-inspired target recognition framework. Our data will be available at https://github.com/DJJJJJJ/SMG_PolSAR .

局限性：尽管该方法在实验中表现优异，但其对DNSP模式的依赖可能限制了在不同极化配置下的泛化能力。此外，混合注意力机制的计算复杂度较高，可能在实际部署中面临效率挑战。

SLIP-RS: Structured-Attribute Language-Image Pre-Training for Remote Sensing Object Detection

SLIP-RS: 结构化属性语言-图像预训练用于遥感目标检测

2026-05-22
arXiv

Chenxu Wang，Yuxuan Li，Yunheng Li，Xiang Li，Jingyuan Xia 等

ID: http://arxiv.org/abs/2605.23144v1

Existing language-image pre-training for remote sensing object detection is constrained by Monolithic Label Learning, which relies on exhaustively enumerating open-set categories via black-box data to acquire fine-grained representations, creating a dependency incompatible with the domain's inherent data scarcity. To transcend this bottleneck, we propose SLIP-RS, establishing a Structured-Attribute Decoupling Paradigm that maps the open-ended category space into a finite, physically meaningful attribute space, unlocking fine-grained discriminability via explicit structural logic. This paradigm is realized via two technical pillars: (1) Structured-Attribute Contrastive Learning, which enforces the learning of decoupled intrinsic visual logic via combinatorial attribute augmentation; and (2) Conformal Attribute Reliability Engine, which leverages conformal prediction theory to rigorously distill high-fidelity supervision from noisy sources, yielding RS-Attribute-15M, the largest dataset with over 15 million attribute annotations. Extensive experiments demonstrate that SLIP-RS establishes unprecedented performance in fine-grained detection and cross-domain generalization, validating structured attributes as a vital foundation for remote sensing.
