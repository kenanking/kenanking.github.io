Hierarchical Identity Learning for Unsupervised Visible-Infrared Person Re-Identification

面向无监督可见光-红外行人重识别的层次化身份学习

2026-07-20
IEEE Transactions on Image Processing (IF: 13.7)

Haonan Shi，Yubin Wang，De Cheng，Lingfeng He，Nannan Wang 等

DOI: 10.1109/tip.2026.3713464

Unsupervised visible-infrared person reidentification (USVI-ReID) aims to learn modality-invariant image features from unlabeled cross-modal person datasets by reducing the modality gap while minimizing reliance on costly manual annotations. Existing methods typically address USVI-ReID using cluster-based contrastive learning, which represents a person by a single cluster center. However, they primarily focus on the commonality of images within each cluster while neglecting the finer-grained differences among them. To address the limitation, we propose a Hierarchical Identity Learning (HIL) framework. Since each cluster may contain several smaller sub-clusters that reflect fine-grained variations among images, we generate multiple memories for each existing coarse-grained cluster via a secondary clustering. Additionally, we propose Multi-Center Contrastive Learning (MCCL) to refine representations for enhancing intra-modal clustering and minimizing cross-modal discrepancies. To further improve cross-modal matching quality, we design a Bidirectional Reverse Selection Transmission (BRST) mechanism, which establishes reliable cross-modal correspondences by performing bidirectional matching of pseudo-labels. Extensive experiments conducted on the SYSU-MM01 and RegDB datasets demonstrate that the proposed method outperforms existing approaches. The source code is available at: https://github.com/haonanshi0125/HIL.

研究结果：在SYSU-MM01和RegDB数据集上的实验表明，HIL方法在无监督跨模态ReID任务中超越了现有方法，显著提升了检索精度。主要意义在于通过细粒度建模和可靠跨模态匹配，有效缓解了聚类中心单一性造成的特征混淆问题，推动了无监督跨模态识别的发展。

ChangeDINO: Bi-level Feature Entanglement Network for Remote Sensing Change Detection with Vision-Language Prior

ChangeDINO：基于视觉-语言先验的双层特征纠缠网络用于遥感变化检测

2026-07-20
IEEE Transactions on Geoscience and Remote Sensing (IF: 8.6)

Yupeng Gao，Tianyu Li，Guoqing Wang，Peng Wang，JiayiMa 等

DOI: 10.1109/tgrs.2026.3715122

The performance of most existing remote sensing change detection (RSCD) methods is often constrained by limited semantic understanding and a heavy dependence on large volumes of training data. To address these limitations, we propose ChangeDINO, a novel change detection paradigm that effectively integrates both semantic knowledge and positional information from Grounding DINO, a representative open-vocabulary detector. To fully leverage Grounding DINO’s semantic guidance and object localization capabilities, we introduce a bi-level feature entanglement strategy, incorporating two key components: the Semantically Guided Fusion Module (SGFM) and the Position-aware Enhancement Module (PEM). The SGFM adaptively exploits the spatially heterogeneous activation patterns of features from the open-vocabulary detector as semantic guidance signals, selectively amplifying the contribution of high-activation object regions while suppressing uninformative background responses. The PEM, in turn, exploits positional embeddings from cross-modal queries and applies feature-wise linear modulation for pixel-level calibration, thereby enhancing spatial precision and refining boundary localization of changed regions. Together, these mechanisms enable ChangeDINO to transfer fine-grained semantics and precise positional cues from large-scale vision-language pre-training into the RSCD task, achieving state-of-the-art performance on standard benchmarks—LEVIR-CD, WHU-CD, and SYSU-CD—as well as on our newly proposed AUUCDD, with F1 scores of 92.89%, 95.79%, 83.87%, and 90.17%, respectively. Furthermore, experiments under data-scarce conditions demonstrate that ChangeDINO maintains superior performance by leveraging Grounding DINO’s rich pretrained representations, thereby reducing dependence on taskspecific annotations and providing a practical solution for data-limited scenarios. The code and datasets are publicly available at https://github.com/G124556/ChangeDINO.

Junming Chen，Bing Liu，Weiqi Lian，Kuiliang Gao，Yue Cao 等

Mixture-of-Experts (MoE) is a popular class of large language models (LLMs), offering high efficiency and accuracy.
