1.5 决策论

概率论为我们提供一个数学框架来定量计算不确定性，而决策论帮助我们在得到不确定性后做出最优决策。

1.5.1 最小化错误分类率

最小化错误分类率的目标就是尽可能少地做出错误分类。

在一个分类问题中，我们寻找一种规则，赋予每一个输入变量\(x\)一个类别。这个规则将输入空间划分为不同的区域\(\mathcal{R}_k\)，每个区域被叫做决策区域（decision region），对应于一个类别。决策区域间的边界被叫做决策边界（decision boundary）或者决策⾯（decision surface）。

最小化错误分类率等价于选择后验概率最大的类别。

1.5.2 最小化期望损失

在某些情况下，最小化错误分类率是不够的。在癌症诊断的例子中，误判主要有两类，正常患者诊断为癌症以及癌症患者诊断为正常。这两类造成的后果是不同的，很明显，我们希望第二种情况发生的尽可能少。

这种情况下，我们用损失函数（loss function），也叫做代价函数（cost function）来描述这个问题，用来度量所有可能决策或动作产生的后果。

在癌症诊断的例子中，我们可以定义损失函数为\(L_{kj}\)（\(k\)表示实际类比，\(j\)表示判断后类别），损失函数以损失矩阵（loss matrix）形式可以表示为： \[ \begin{array}{lcc} & \text { 癌症 } & \text { 正常 } \\ \text { 癌症 } & 0 & 1000 \\ \text { 正常 } & 1 & 0 \end{array} \]

于是我们可以得到模型关于联合分布\(p\left(\mathbf{x}, \mathcal{C}_{k}\right)\)的平均损失：

\[ \mathbb{E}[L]=\sum_{k} \sum_{j} \int_{\mathcal{R}_{j}} L_{k j} p\left(\mathbf{x}, \mathcal{C}_{k}\right) \mathrm{d} \mathbf{x} \]

使上式最小化，等价于对于每个\(\mathbf{x}\)，我们要最小化\(\sum_{k} L_{k j} p\left(\mathcal{C}_{k} \mid \mathrm{x}\right)\)。

于是，最小化期望损失等价于选择使\(\sum_{k} L_{k j} p\left(\mathcal{C}_{k} \mid \mathrm{x}\right)\)最小的第\(j\)类。

1.5.3 拒绝选项

当后验概率\(p\left(\mathcal{C}_{k} \mid \mathbf{x}\right)\)远小于\(1\)时，做出错误分类的概率就会提升。在某些应用中，对于决策困难的情况，我们需要避免做出决策，这种情况称为拒绝选项（reject option）。

做法是设定一个阈值\(\theta\)，拒绝后验概率小于等于\(\theta\)的输入。

1.5.4 推断和决策

分类问题可以分为两个阶段：推断（inference）阶段和决策（decision）阶段。

一种方法是：在推断阶段，使⽤训练数据学习\(p\left(\mathcal{C}_{k} \mid \mathrm{x}\right)\)的模型。在决策阶段，使⽤后验概率进行最优的分类。

另⼀种⽅法是：同时解决两个问题，即学习一个函数，直接将输入映射为决策，这个函数称为判别函数（discriminant function）。

事实上在决策问题中有三个模型，复杂度由高到低为：

  1. 生成模型（generative model）

    显式地或隐式地对输⼊以及输出的分布进行建模的⽅法。

  2. 判别模型（discriminative model）

    直接对后验概率建模的方法。

  3. 找到⼀个函数\(f(\mathrm{x})\)，称为判别函数（discriminant function）。这个函数把每个输入\(\mathbb{E}[L]=\iint L(t, y(\mathbf{x})) p(\mathbf{x}, t) \mathrm{d} \mathbf{x} \mathrm{d} t\)直接映射为类别标签。

1.5.5 回归问题的损失函数

在回归问题中，对于每一个输入\(\mathrm{x}\)及其对应的\(t\)，估计一个函数\(y(\mathrm{x})\)，对应的损失函数为\(L(t, y(\mathbf{x}))\)。可以得到期望损失为： \[ \mathbb{E}[L]=\iint L(t, y(\mathbf{x})) p(\mathbf{x}, t) \mathrm{d} \mathbf{x} \mathrm{d} t \]

在回归问题中，常用的损失函数是平方损失函数：\(L(t, y(\mathbf{x}))=\{y(\mathbf{x})-t\}^{2}\)。对应期望损失可以写为： \[ \mathbb{E}[L]=\iint\{y(\mathbf{x})-t\}^{2} p(\mathbf{x}, t) \mathrm{d} \mathbf{x} \mathrm{d} t \]

目标是使得期望损失最小，通过变分法求解： \[ \frac{\delta \mathbb{E}[L]}{\delta y(\mathbf{x})}=2 \int\{y(\mathbf{x})-t\} p(\mathbf{x}, t) \mathrm{d} t=0 \]

求解得到\(y(\mathrm{x})\)： \[ y(\mathrm{x})=\frac{\int t p(\mathrm{x}, t) \mathrm{d} t}{p(\mathrm{x})}=\int t p(t \mid \mathrm{x}) \mathrm{d} t=\mathbb{E}_{t}[t \mid \mathrm{x}] \]

\(y(\mathrm{x})\)是目标变量\(t\)在条件\(\mathrm{x}\)下的条件均值，这个函数称为回归函数（regression function）。

这个结果也可以通过一种稍微不同的方式推出。我们可以把期望损失中的平方项按照下面的方式展开： \[ \begin{array}{l} \{y(\mathbf{x})-t\}^{2}=\{y(\mathbf{x})-\mathbb{E}[t \mid \mathbf{x}]+\mathbb{E}[t \mid \mathbf{x}]-t\}^{2} \\ \quad=\{y(\mathbf{x})-\mathbb{E}[t \mid \mathbf{x}]\}^{2}+2\{y(\mathbf{x})-\mathbb{E}[t \mid \mathbf{x}]\}\{\mathbb{E}[t \mid \mathbf{x}]-t\}+\{\mathbb{E}[t \mid \mathbf{x}]-t\}^{2} \end{array} \]

当计算期望损失时，中间的交叉项消失： \[ \mathbb{E}[L]=\int\{y(\mathbf{x})-\mathbb{E}[t \mid \mathbf{x}]\}^{2} p(\mathbf{x}) \mathrm{d} \mathbf{x}+\int\{\mathbb{E}[t \mid \mathbf{x}]-t\}^{2} p(\mathbf{x}) \mathrm{d} \mathbf{x} \]

上式中，第一项最小等价于最小二乘的最优解为条件均值，这个结论与之前得出的是一样的。第二项是\(t\)的分布的方差在\(p(\mathbf{x})\)上的均值，可以看作噪声。

在回归问题中，平方损失函数不是唯一的选择。在某些情况下会用到闵可夫斯基损失函数（Minkowski loss），它是平方损失函数的推广。对应期望损失为： \[ \mathbb{E}\left[L_{q}\right]=\iint|y(\mathbf{x})-t|^{q} p(\mathbf{x}, t) \mathrm{d} \mathbf{x} \mathrm{d} t \]

当\(q = 2\)时，这个函数就变成了平⽅损失函数的期望损失。
